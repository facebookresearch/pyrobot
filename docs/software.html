Installing the robot software

PyRobot 0.1 is currently only supported on Ubuntu 16.04.

Installing PyRobot and LoCoBot

If you are trying to set up both PyRobot and LoCoBot dependencies, please follow the instructions in the LoCoBot repository https://github.com/adithyamurali/LoCoBot

Installing just PyRobot

  • Install Ubuntu 16.04

  • Install ROS kinetic

  • Install KDL

sudo apt-get -y install ros-kinetic-orocos-kdl
sudo apt-get -y install ros-kinetic-kdl-parser-py
sudo apt-get -y install ros-kinetic-python-orocos-kdl
  • Install Python virtual environment
sudo apt-get -y install python-virtualenv
virtualenv_name="pyenv_pyrobot"
VIRTUALENV_FOLDER=~/${virtualenv_name}
virtualenv --system-site-packages -p python2.7 $VIRTUALENV_FOLDER
  • Install PyRobot
cd ~
source ~/${virtualenv_name}/bin/activate
git clone https://github.com/adithyamurali/pyrobot.git
cd pyrobot
pip install .
